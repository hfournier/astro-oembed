## Keeping the Provider Snapshot up to date

The list above reflects the **Provider Snapshot** bundled with the installed version of `astro-oembed` — the same data the `<Oembed>` component uses at runtime. To pull in newly registered providers from the [oEmbed registry json file](https://oembed.com/providers.json), run:

```sh
pnpm update-providers
```

from the package root, then release a new version.
